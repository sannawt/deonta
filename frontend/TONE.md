# Provisional Tone

Use provisional language in every user-facing explanation.

| Avoid | Prefer |
|------|--------|
| Applies / Does not apply | Indicates in scope / Indicates out of scope |
| Pass | Appears met / in force / none asserted |
| Verdict | Provisional indication / Preliminary finding |
| Reasoning trace | How this was reached, on your facts |
| Adding X flips Y | If X were added, the indication would change to Y |
| Defeasibility | This holds unless... |
| Facts in evidence | Facts you've provided |
| The engine determined | On those facts, the rules would treat... |
| Compliance check | Compliance scoping |

Headlines should begin with a phrase equivalent to:

> On the facts you've provided...

The symbolic engine remains the source of truth for legal conclusions. UI or
narration layers may explain or summarize, but must not change the result.
